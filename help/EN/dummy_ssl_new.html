The Dummy SSL certificate plug-in is designed to facilitate manual SSL certificates provisioning and/or SSL certificate procedures testing. Actually the plug-in emulates Certificate authority activities.

To configure the plug-in, fill the form:

Field Name
Description
Process auto renewal before SSL certificate expiry date (days) Enter how many days before SSL certificate expiration the plug-in should request auto renewal.
Delay checking of SSL Certificate status (days, 0 = immediately) Enter how many days the plug-in should wait before it starts checking SSL certificate status. Note that if 0 days is entered, then the plug-in starts checking right after the certificate issue request is sent.
Duration of checking SSL certificate availability (days) Enter how many days the plug-in should keep checking the SSL certificate status. By default, the plug-in checks the certificate status every hour. The certificate status is updated automatically.
SSL certificate mode Select the mode of SSL certificate generation:
  • No operation. The Dummy plug-in will not actually create a certificate.
  • Fake certificate. The plug-in will create some text instead of a certificate, for example, "TEST CERTIFICATE".
  • Self-signed. The Dummy plug-in will create a self-signed certificate.
