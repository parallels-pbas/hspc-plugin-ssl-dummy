Dummy SSL Certificate Plug-In

The Dummy SSL certificate plug-in is designed to facilitate manual SSL certificates provisioning and/or SSL certificate procedures testing. Actually the plug-in emulates Certificate authority activities.

The following SSL certificate types are implemented in the Dummy SSL certificate plug-in:

  • No certificate is actually generated.
  • Fake certificate. Some text is created instead of a certificate, for example, "TEST CERTIFICATE".
  • Self-signed SSL certificate is generated.
